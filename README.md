# capstone-2-
Business Analysis with Python - EmporiUm Sales Analysis

## Project Description
This project analyzes in-store sales data for EmporiUm, a growing student 
bookstore chain that sells tech gear, art supplies, and other products.

## Objective 
To analyze EmporiUm's in-store sales data for Maryland and Massachusetts 
territories and provide the marketing team with data-driven insights and 
actionable recommendations for the next quarter.

## Sales Territories Analyzed
- **Territory 1: Maryland** — Territory Manager: Shruti Reddy (9 stores)
- **Territory 2: Massachusetts** — Territory Manager: Bo Heap (18 stores)
- **Region:** Northeast
- **Region Director:** Michael Jarvis

## key Analysis Areas 
- Territory managers, store IDs, and store locations for each territory
- Monthly total revenue for in-store sales in each territory
- Sales performance ranking of each store in each territory
- Top customers in each territory based on rewards membership
- Number of transactions and total revenue per month by product category
- Marketing recommendations for the next quarter
- Data visualizations using Matplotlib charts

## Files Included
- `teklemariam_sales_analysis.ipynb` — Main analysis notebook
- `StoreSales.csv` — Transaction data
- `StoreDetail.csv` — Store location and territory information
- `Products.csv` — Product details
- `ProductCategories.csv` — Product categories and subcategories
- `customer_list.csv` — Rewards member information

## Tools Used
- Python, pandas, NumPy, Matplotlib
- Jupyter Notebook, GitHub
